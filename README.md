# EXPO2021
Proyecto de la EXPO 2021, cuyo nombre fue Young Mental Health EXPO. Estuvo enfocado en crear un sitio web con tips o métodos para ayudar a tener una mejor salud mental. El nombre de nuestra organización (o grupo) es SavvyTech, nuestro objetivo fue crear un foro, para que los usuarios pudieran compartir tips y experiencias; así otros podrán aprender de ello.

Aquí se podrá encontrar adjunto el link para la página web funcional (desde que escribo esto, 6 de junio de 2022) y el vídeo mostrando el proyecto a una audiencia.
